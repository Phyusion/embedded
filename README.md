# Presentations

A gallery of presentation slideshows, hosted via GitHub Pages.

**Gallery URL:** https://phyusion.github.io/embedded/

## Structure

```
├── index.html                  ← Gallery page listing all presentations
└── simulators-dilemma/
    └── index.html              ← Individual slideshow (self-contained HTML)
```

## Adding a new presentation

1. Create a new folder: `mkdir my-talk-name`
2. Drop the self-contained HTML slideshow into it as `index.html`
3. Add a card to the root `index.html`
4. Commit and push — done

The embed URL for Substack/blogs is:
`https://phyusion.github.io/embedded/my-talk-name/`
